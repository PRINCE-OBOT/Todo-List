# Todo List

A task management app where users can create, organize, and track daily tasks and subtasks.

Built as part of [The Odin Project](https://www.theodinproject.com/lessons/node-path-javascript-todo-list) curriculum to practice JavaScript application architecture, DOM manipulation, modular code organization, and state management.

---

## Features

* Create and delete tasks
* Add subtasks to tasks
* Mark tasks and subtasks as completed
* Organize tasks by project
* Dynamic DOM rendering
* Responsive user interface
* Local storage persistence
* Modular JavaScript architecture

---

## Tech Stack

* JavaScript
* HTML
* CSS
* Webpack

---

## Task Management

### Tasks

* Create tasks
* Edit task details
* Delete tasks
* Track completion status

### Subtasks

* Add subtasks to tasks
* Mark subtasks as completed
* Organize smaller objectives under main tasks
